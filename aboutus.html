About the Developer

Hi! I'm Pushpavathi Javvadi, a student in ISM 6225 - App Dev for Analytics. This project helped me understand how to design, build, and host a user-friendly static website using front-end technologies like HTML, CSS, and JavaScript.

Pushpa Travel Agency was created to simulate a real travel agency website. Through this project, I gained experience in user interface layout, form design, and data visualization using Chart.js. It improved my skills in organizing code, creating reusable components, and deploying content to a live server.

Technologies Used

  • HTML5 – structure and content of the site
  • CSS3 – styling and layout of all pages
  • JavaScript – basic interactivity (like alert or form check)
  • Chart.js – visualizing travel booking data
  • GitHub – version control and code hosting
  • Cyberduck + MyWeb – website deployment
